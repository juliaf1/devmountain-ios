Self-Assessment Shopping List

Please watch the screen recording provided in the download folder before starting this self-assessment. It demonstrates the application you are to create.

In this self-assessment, you will build a shopping list application. The shopping list should allow you to create shopping items, as well as check them off if they have been purchased.

Instructions

  1. The initial screen should show a list of shopping items. Each row shows the item title and a checkbox (checked if purchased, unchecked if not).

  2. There should be an add button that will allow the user to add shopping list item to the list of shopping items. This can be done by using a text field on the table view controller. It can also be done with an alert controller.

  3. Tapping on a checkbox will toggle the items purchased state.

  4. Quitting the app and re-launching it will show your shopping items and their purchased state (persistence).

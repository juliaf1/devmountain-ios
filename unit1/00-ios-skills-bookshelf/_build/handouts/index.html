Self-Assessment Bookshelf

In this self-assessmet you will attempt to build a bookshelf application. Instructions are provided below, as well as a demo video. Have fun with this application and add your own personal touches. As this is a self-assessment, push yourself to complete this app without assistance, however, if you cannot move past an issue, reach out for support.

Instructions

Please download and watch the screen recording provided before starting. It demonstrates the application you are to create.

The bookshelf should hold 6 books of your choosing. You will need to go online and get the appropriate information for each book.

Note the following requirements:

  1. The initial screen shows a list of books. Each row shows the book title and the authorâ€™s name.

  2. Tapping on a row will take you to the books detail page.

  3. The detail view should present the books details including: title, author, cover, release year, and description.

  4. The detail view should include a back button that, when pressed, takes you back to the initial screen.

Going Further: - Use the Date type for the release date and present the full date to the user (month, day, year).
